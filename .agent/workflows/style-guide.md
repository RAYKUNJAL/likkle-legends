# Likkle Legends Visual Identity Rules

These rules are MANDATORY for all code and asset generation tasks involving the Likkle Legends brand.

## 1. Imagery Style
- **Hero/Lifestyle Images**: MUST use high-quality photography of **real people**. Never use cartoons or 3D-generated characters for hero/lifestyle shots unless they are the specific Branded Characters.
- **Vibe**: Tropical, premium, commercial-grade.

## 2. Character Consistency (The Legends)
All images of characters must strictly follow their established web app designs. Do not "re-imagine" these characters outside of their existing branding.

### The Core Legends:
- **Tanty Spice**: Warm, wise, vibrant Caribbean mother figure.
- **R.O.T.I.**: Friendly reading robot with specific scholarship theme.
- **Dilly Doubles**: Energetic, purple-themed.
- **Steelpan Sam**: Musical, blue-themed.
- **Mango Moko**: Stilted adventurer, green-themed.
- **Scorcha Pepper**: The mischievous red villain.

## 3. Maintenance
- **Cleanup**: Remove or replace any cartoonish placeholders that represent humans.
- **Source of Truth**: The `/public/images` directory contains the official character assets.
