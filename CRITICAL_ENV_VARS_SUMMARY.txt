CRITICAL CONFIGURATION RULES
================================================================================

1. All [Secret] variables MUST be marked as "Secret" in Vercel UI
   - SUPABASE_SERVICE_ROLE_KEY
   - PAYPAL_CLIENT_SECRET
   - GOOGLE_GENERATIVE_AI_API_KEY or GEMINI_API_KEY
   - RESEND_API_KEY or BREVO_API_KEY

2. All [Public] variables MUST start with "NEXT_PUBLIC_"
   - NEXT_PUBLIC_SUPABASE_URL
   - NEXT_PUBLIC_SUPABASE_ANON_KEY
   - NEXT_PUBLIC_PAYPAL_CLIENT_ID
   - All NEXT_PUBLIC_PAYPAL_PLAN_* variables

3. PayPal plan IDs MUST be in format: P-XXXXXXXXXXXXX
   - No empty values allowed
   - All 8 plan variables required

4. No variable should be empty or truncated
   - Check Vercel UI shows full values
   - Copy/paste carefully to avoid whitespace

================================================================================
WHERE TO SET THESE ON VERCEL
================================================================================

1. Go to https://vercel.com
2. Click on project: "likkle-legends"
3. Go to: Settings → Environment Variables
4. For each variable:
   - Enter variable name (e.g., NEXT_PUBLIC_SUPABASE_URL)
   - Enter value (paste from secure location)
   - Select production environment (if available)
   - Mark as Secret if indicated above
   - Click "Save"
5. After ALL variables are added, trigger redeploy

================================================================================
WHAT HAPPENS IF VARIABLES ARE MISSING
================================================================================

Missing SUPABASE variables:
  → App cannot start, database connection fails
  → ERROR: [server-env] Missing required environment variable

Missing PAYPAL variables:
  → Checkout page broken, subscription system unavailable
  → ERROR: PayPal configuration incomplete

Missing GEMINI/GOOGLE_GENERATIVE_AI variables:
  → All AI features fail (buddy chat, games, stories, image generation)
  → ERROR: GEMINI_API_KEY is missing in environment

Missing EMAIL service variables:
  → Users cannot receive signup emails, password resets fail
  → ERROR: No email service configured
